# Carcassonne AI Pygame Framework

This repository contains a simple Pygame framework with a menu UI to serve as the starting point for a Carcassonne AI project.

## Features

- Basic window setup using Pygame
- Menu with `Start Game` and `Quit` options
- Placeholder `Game` class for later development
- Requirements file for easy setup

## Getting Started

1. Create a Python virtual environment and activate it.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```
4. Use the arrow keys to navigate the menu and `Enter` to select.

## Project Structure

```
carcassonne_AI/
├── assets/          # image assets
├── game.py          # game logic placeholder
├── main.py          # entry point with loop and menu
├── menu.py          # simple menu UI
├── requirements.txt
└── README.md
```

Feel free to expand the `Game` class with AI logic and board rendering.
